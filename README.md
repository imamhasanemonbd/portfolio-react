## 🚀 Key Highlights & Architectural Features

*   **⚡ Next-Gen Glitch Engine**: A custom, high-fidelity React-ported monochromatic glitch effect (`mgGlitch.js` concept) driving the hero background experience.
*   **📐 Premium Micro-Layouts**: Hand-crafted, responsive grids with elegant glassmorphism nodes, custom floating mobile arrow actions, and inertia cursor tracking.
*   **🎨 Harmoneous Grayscale Mapping**: Highly optimized grayscale maps integration (centered on Feni, Bangladesh) maintaining visual consistency with the dark monochrome style.
*   **📱 Single-Line Social Alignment**: Advanced CSS flex configurations fitting all 6 major social connections into a single line across desktop and mobile.
*   **🧹 Ultra-Clean Footprint**: Zero template leftovers. Fully audited codebase with all dead template sections, unused states, and obsolete assets completely purged.

## ⚙️ Getting Started & Local Development

### 1. Installation
Clone the repository and install all node modules:
```bash
git clone https://github.com/imamhasanemonbd/portfolio-react.git
cd portfolio-react
npm install
```

### 2. Launch Local Dev Server
Fire up the Next.js development bundle:
```bash
npm run dev
```
Open **[http://localhost:3000](http://localhost:3000)** in your browser to view the application.

### 3. Production Compilation
Verify compilation and optimize assets:
```bash
npm run build
npm run start
```
